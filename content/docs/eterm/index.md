ETerm is a terminal for AI CLI workflows. Currently available on macOS.

We built it for ourselves — we wanted a better experience working with AI in the terminal. If you're looking for something similar, it might help.

## Get Started

Download the latest release from [GitHub Releases](https://github.com/vimo-ai/ETerm/releases).

Want to build from source? See the [Installation Guide](/docs/eterm/installation).
